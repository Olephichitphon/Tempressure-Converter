A sleek, real-time temperature conversion tool built with Vanilla JavaScript. This project allows users to instantly convert between Celsius, Fahrenheit, and Kelvin as they type. It features precise decimal calculations and a responsive interface.

### Features
- **Real-time Conversion:** Values update instantly using the `input` event.
- **Precision:** All results are rounded to 2 decimal places for accuracy.
- **Auto-reset:** Clears all fields automatically when input is deleted.
- **Clean Code:** Implemented with modular logic and optimized event listeners.
